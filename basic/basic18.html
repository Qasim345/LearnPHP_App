PHP Float

This data type represents decimal values. The float (floating point number) is a number with a decimal point or a number in exponential form.
Syntax 

$a=1.234;  
$x=1.2e4;  
$y=7E-10;

Example 1

<?php  
    $x=22.41;  
    echo $x;  
?>
Output: 22.41

Example 2

<?php   
          $a = 11.365;  
          var_dump($a);  
      ?>
Output: 11.365

Example 3

<?php  
    $a = 6.203;  
    $b = 2.3e4;  
    $c = 7E-10;  
    var_dump($a);  
    var_dump($b);  
    var_dump($c);  
?>
Output: float 6.203
float 23000
float 7.0E-10
