Even Odd Program

Logic:

  • Take a number.
  • Divide it by 2.
  • If the remainder is 0, print number is even.

Program

<?php  
$number=1233456;  
if($number%2==0)  
{  
 echo "$number is Even Number";   
}  
else  
{  
 echo "$number is Odd Number";  
}   
?>
Output: 1233456 is Even Number
